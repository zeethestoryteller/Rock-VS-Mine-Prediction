<h1> Rock vs. Mine Prediction using SONAR Data </h1><br>
This project implements a Machine Learning model to identify whether an underwater object is a Rock or a Naval Mine based on sonar return signatures. <br>

<h1>Project Overview</h1>
The dataset contains sonar data obtained by bouncing sonar signals off various surfaces. A metal cylinder (mine) returns a different frequency pattern compared to a rock. This project uses Logistic Regression, a statistical method for binary classification, to predict the category of the object.
